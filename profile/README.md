## What is SuperML?

**SuperML.dev** is an applied AI lab for developers, builders, and curious engineers who want more than theory.

We create practical, open-source AI projects that help you:

- learn from real use cases
- understand how systems work end to end
- reuse starter code and templates
- move from notebooks to usable products

This is not AI for hype.  
This is AI for people who want to **build**.

## Focus Areas

### Agentic AI
Hands-on projects for autonomous and semi-autonomous AI workflows, orchestration patterns, and developer-first agent systems.

### Machine Learning with Real Data
Reproducible projects using financial data, anomaly detection, and practical modeling pipelines.

### Fraud & Risk Intelligence
Experiments and frameworks for transaction monitoring, anomaly detection, and explainable risk scoring.

### Voice & Media AI
Projects that explore synthetic voice, video generation, and multimodal AI experiences.

### Open Learning
Guides, templates, and examples designed to make AI implementation easier for developers.
